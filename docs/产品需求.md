
# 国际医学部护理管理系统

## 文档状态

本文描述系统总体需求，包含一期排班管理和后续绩效、奖金扩展能力。当前执行计划优先实现一期排班管理：人员/班次/节假日维护、月排班表、日期与周查询、快速填班、全屏、打印、周统计、加班班次与总系数核算。月核算、奖金分配、完整账号体系和 etcd 正式存储作为后续扩展，不进入一期实现。

## 排班管理系统功能需求

该系统的核心需求可归纳为以下四个方面：

### 1. 人员与班次配置管理
*   **人员信息管理**：系统需预设护士、文员等人员名单及对应的工号（如护士长段鸿露 000228）。
*   **班次类型定义**：支持设置**常用班次**（如A1组长、P1、N1、办公班等）及**不常用班次**（如进修、病假、产假等）。
*   **绩效系数设定**：为不同岗位和班次设定核算系数。例如，护士长系数为1.8，组长系数为1.5，普通班次为1.3，办公班为1.2，辅班为0.5等。

### 2. 交互式可视化排班
*   **多维选择与显示**：支持按年、月、日、周切换视图。
*   **快速排班操作**：提供班次选择面板，支持**点击即输入**的快速填充功能。
*   **视觉辅助设计**：
    *   表格顶部需清晰显示日期和星期。
    *   **周末及法定节假日**（需显示节日名称）应以**红色**标注。
    *   特殊状态显示：如“夜班”或“请假”等特殊状态可以显示为特定颜色（如红色）以便区分。
*   **界面功能**：支持全屏显示、排班查询以及排班表的打印预览与输出。

### 3. 绩效与出勤自动核算逻辑
*   **满勤判定规则**：常规每周5天为满勤。若遇法定节假日（不调休），满勤天数相应扣减（如端午节当周4天即为满勤）。
*   **出勤盈亏统计**：根据实际出勤天数与满勤标准的差值自动计算，例如出勤4天记为-1，5天为0，6天为+1。
*   **多维度统计**：
    *   **周汇总**：每周五自动汇总每人本周的出勤总天数及总系数。
    *   **月核算**：每月最后一天自动汇总全月的出勤总班次、总天数及总系数。

### 4. 奖金分配功能
*   **自动化分配**：系统提供总金额输入口，根据每人每月的**总班次及总系数**，自动按比例核算并生成个人奖金数额。
